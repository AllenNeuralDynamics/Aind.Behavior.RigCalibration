DataMapper
-------------

The `data_mapper` module of this library is used as an adapter interface between the (meta)data generated by this library and other AIND services, namely `aind-data-schema <https://github.com/AllenNeuralDynamics/aind-data-schema>`_.

The design and pattern of this module will likely evolve over time, but for now it provides a simple, function-oriented interface to generate :py:class:`~aind-data-schema.core.session.Session`, the required data object to interact with AIND services:

:py:class:`~aind_behavior_services.aind_services.launcher.data_mapper_service.map`

or via the syntactic sugar methods:
:py:class:`~aind_behavior_services.aind_services.launcher.data_mapper_service.map_from_session_root` and  :py:class:`~aind_behavior_services.aind_services.launcher.data_mapper_service.map_from_json_files`

For an implementation example see the :py:class:`~aind_behavior_services.launcher.Launcher` class.
